# Echo Lite Starter

Starter project Golang + Echo + JWT + PostgreSQL + Zerolog + .env + Migrate

## Fitur
- PostgreSQL tanpa ORM
- Error handling terpusat
- Logging dengan zerolog
- Migrasi pakai golang-migrate

## Setup

1. Copy `.env`
2. Jalankan migrate:
   make migrate-up
3. Jalankan server:
   make run
